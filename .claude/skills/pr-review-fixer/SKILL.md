# Skill: pr-review-fixer

## 실행 절차

### Step 1: 코멘트 분류
- **즉시 수정 가능**: 코드 변경으로 해결
- **논의 필요**: 방향성/설계 결정 필요
- **이해 불가**: 추가 설명 필요

### Step 2: 수정 적용 (즉시 수정 가능)
```
[ ] 변경 파일 목록 작성
[ ] 각 코멘트별 수정 적용
[ ] 수정 후 테스트 실행
[ ] 코멘트별 완료 체크
```

### Step 3: 미적용 항목 정리
```markdown
## 논의 필요: [코멘트] — [논의 이유]
## 질문: [코멘트] — [명확화 필요한 부분]
```

## 완료 기준
```
[ ] 즉시 수정 가능한 모든 코멘트 반영
[ ] 미적용 항목 정리 완료
[ ] 테스트 통과
```
