## 무엇을 만드는가

리뷰 래퍼 `plugins/rein-core/scripts/rein-codex-review.sh` 에 두 기능:
- **A 무상태 자가검증 관문**: 모든 비면제 코드리뷰 호출 직전 로컬 검증 증거(타입체크 exit0 + 테스트 exit0 두 축, 또는 검증명령 없음 폴백, 또는 TDD red-phase escape) 강제. 부재 시 codex 호출 이전 `exit 4` + 앵커 `ERROR: [codex-review][readiness-reject]`. 목적 = "리뷰 지적 고침 → 새 문제 → 재리뷰" 회귀 루프 차단.
- **C 통과답변 출력축소**: envelope 지시를 "통과 항목 요약 + 발견만 상세", `FINAL_VERDICT` 계약 불변, `REIN_REVIEW_VERBOSE=1` escape, 네 검사 지시문 존재 정적 검증.
- 면제(B: 산문/주석/버전)는 이번 사이클 제외 → 별도 후속 사이클(각 독자 위험모델).
